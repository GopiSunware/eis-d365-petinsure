Quick Start Commands

Claims Data API (Port 8000)

cd src/shared/claims_data_api && uvicorn app.main:app --port 8000 --reload

Agent Pipeline (Port 8006)

cd src/ws6_agent_pipeline && uvicorn app.main:app --port 8006 --reload

Agent Portal UI (Port 8080)

cd src/ws4_agent_portal && npm run dev # Runs on :8080

Admin Portal UI (Port 8081)

cd src/ws8_admin_portal/frontend && npm run dev # Runs on :8081
